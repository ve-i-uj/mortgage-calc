Ипотечный калькулятор с графическим интерфейсом. 

Калькулятор предназначен для учета и планирования платежей в случае 
ипотеки с аннуитетным платежом и возможностью частичного 
досрочного погашения (например, Сбербанк).

Возможности калькулятора:

1. с помощью калькулятора можно планировать досрочное погашение кредита, т.е.:
    - по желаемой дате досрочного погашения расчитывает 
      необходимую для этого сумму ежемесячного платежа;
    - и наоборот, по сумме ежемесячного платежа высчитает дату окончания кредита.
    (планируемую дату/платеж он высчитывает, учитывая уже имеющуюся историю платежей)

2. калькулятор хранит историю платежей вместе с дополнительной информацией, такой как:
    - суммой взноса по кредиту
    - сколько банк взял за пользование кредитом
    - сумма ежемесячного платежа
    - остаток кредита
    - переплата
    - выгода от переплаты

3. калькулятор учитывает в общей ипотечной истории вклад каждого плательщика,
   (если плательщиков несколько и каждый делает отдельные взносы)

4. калькулятор предупреждает, если следующий платеж попадает 
   на выходные (во время внесения информации о последнем платеже)

5. показывает диаграмму платежей, переплат, опережение банковского графика платежей.


Калькулятор тестировал на Debian 7 (рабочая среда Xfce) и 
Ubuntu 10.04 - выглядит всё нормально.

В  Windows XP (виртуальная машина) - работает, но 
выглядит не столь симпатично.
Насчёт работы в других ОС - не знаю.


Установка:

1. перейти в консоле в дирикторию с исходными кодом
2. python3 setup.py install


Вместе с исходными тексами идет файл Demo.clc - это demo-история платежей
для просмотра возможностей калькулятора.
